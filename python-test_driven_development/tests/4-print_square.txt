The ``print_square`` function prints a square made of '#'.

Import:

    >>> print_square = __import__('4-print_square').print_square

Size 1:

    >>> print_square(1)
    #

Size 0 prints nothing (no output):

    >>> print_square(0)

Size 2:

    >>> print_square(2)
    ##
    ##

Negative size:

    >>> print_square(-1)
    Traceback (most recent call last):
    ValueError: size must be >= 0

Type errors:

    >>> print_square("4")
    Traceback (most recent call last):
    TypeError: size must be an integer

A float (even if it's .0) is not allowed:

    >>> print_square(3.0)
    Traceback (most recent call last):
    TypeError: size must be an integer
