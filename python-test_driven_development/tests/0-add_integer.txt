The ``add_integer`` function adds two numbers (integers or floats).

Import the function:

    >>> add_integer = __import__('0-add_integer').add_integer

Basic additions:

    >>> add_integer(1, 2)
    3
    >>> add_integer(100, -2)
    98

Default value for b:

    >>> add_integer(2)
    100

Floats are casted to integers:

    >>> add_integer(100.3, -2)
    98
    >>> add_integer(3.9, 1.2)
    4

Type checking for b:

    >>> add_integer(4, "School")

    TypeError: b must be an integer

Type checking for a:

    >>> add_integer(None)
    Traceback (most recent call last):
    TypeError: a must be an integer
