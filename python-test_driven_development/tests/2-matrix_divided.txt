The ``matrix_divided`` function divides all elements of a matrix.

Import:

    >>> matrix_divided = __import__('2-matrix_divided').matrix_divided

Basic example and immutability:

    >>> matrix = [[1, 2, 3], [4, 5, 6]]
    >>> matrix_divided(matrix, 3)
    [[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]
    >>> matrix
    [[1, 2, 3], [4, 5, 6]]
