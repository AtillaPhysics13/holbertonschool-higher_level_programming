The ``matrix_divided`` function divides all elements of a matrix.

Import:

    >>> matrix_divided = __import__('2-matrix_divided').matrix_divided

Basic example and immutability (original matrix must not change):

    >>> matrix = [[1, 2, 3], [4, 5, 6]]
    >>> matrix_divided(matrix, 3)
    [[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]
    >>> matrix
    [[1, 2, 3], [4, 5, 6]]

Matrix must be a list of lists of integers/floats:

    >>> matrix_divided("not a matrix", 3)
    Traceback (most recent call last):
    TypeError: matrix must be a matrix (list of lists) of integers/floats

Each row must have the same size:

    >>> matrix_divided([[1, 2], [3, 4, 5]], 2)
    Traceback (most recent call last):
    TypeError: Each row of the matrix must have the same size

div must be a number and cannot be zero:

    >>> matrix_divided([[1, 2], [3, 4]], 0)
    Traceback (most recent call last):
    ZeroDivisionError: division by zero
