The ``matrix_divided`` function divides all elements of a matrix.

Import:

    >>> matrix_divided = __import__('2-matrix_divided').matrix_divided

Basic example and immutability (original matrix must not change):

    >>> matrix = [[1, 2, 3], [4, 5, 6]]
    >>> matrix_divided(matrix, 3)
    [[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]
    >>> matrix
    [[1, 2, 3], [4, 5, 6]]

Matrix must be a matrix (list of lists) of integers/floats:

    Traceback (most recent call last):
    ZeroDivisionError: division by zero

    >>> matrix_divided([[1, 2], [3, 4]], "2")

Infinity / NaN behavior (should be accepted as floats):

    >>> matrix_divided([[1, 2], [3, 4]], float('inf'))
    [[0.0, 0.0], [0.0, 0.0]]

    >>> res = matrix_divided([[1, float('inf')]], 2)
    >>> res[0][0]
    0.5
    >>> res[0][1]
    inf

    >>> res = matrix_divided([[1, float('nan')]], 2)
    >>> res[0][0]
    0.5
    >>> str(res[0][1])
    'nan'
